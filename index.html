Visualizing the ARKeth Blockchain Transactions

What is this?

This is a way of plotting the behavior of many simultaneous transactions at once. Each circle corresponds to a set of transactions, and the size and color of the circle is determined by how long the transaction on the ARKeth simulation takes to execute. If the transaction does not complete in the allotted time frame, then we color the circle black.

Transactions are generated as random simulated DAG Tangle transaction, since any multiplex transaction is valid Tangle/Block chain transaction. We map these transaction into a 2D plane using the Hilbert Curve.

The Tangle Transaction space starts small but increases over time. Since longer taking transactions can take longer to complete, this is why the circles get larger over time.
